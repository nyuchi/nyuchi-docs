## Companion site

The Bundu Foundation's outward-facing docs — the Mzizi design system itself,
the Ubuntu doctrine, the Bundu brand — live at
[docs.bundu.org](https://docs.bundu.org). This site is the inward-facing
engineering counterpart: how Nyuchi builds and operates, and how to consume
those Bundu projects from a Nyuchi codebase.
